# 📌 To-Do App (Django)

A modern and efficient **To-Do List Web Application** built using the **Django framework**. This project helps users manage daily tasks, stay organized, and improve productivity through a simple and user-friendly interface.

---

## 🚀 Features

- ✨ Add, update, and delete tasks  
- 📋 Organize tasks efficiently  
- ✅ Mark tasks as completed or pending  
- 🔐 Clean and intuitive user interface  
- ⚡ Fast and responsive performance  
- 🗂️ Well-structured Django project  

---

## 🛠️ Tech Stack

- 🐍 Python  
- 🌐 Django  
- 🎨 HTML, CSS  
- 🗄️ SQLite  

---

## 🎯 Project Objective

The purpose of this project is to demonstrate the implementation of **CRUD operations (Create, Read, Update, Delete)** using Django while building a practical task management application.

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Laraib-Qandeel/To_Do_App_via_Djngo

# Navigate to the project directory
cd To_Do_App_via_Djngo

# Create a virtual environment
python -m venv env

# Activate the virtual environment
# Windows:
env\Scripts\activate

# Mac/Linux:
source env/bin/activate

# Install dependencies
pip install -r requirements.txt

# Apply migrations
python manage.py makemigrations
python manage.py migrate

# Run the development server
python manage.py runserver
